# Error Tracking (LogRocket)

LogRocket session replay and error tracking integration.

## Setup

1. Sign up at [LogRocket](https://logrocket.com)
2. Get your App ID from the dashboard
3. Add to `.env`:
   ```
   NEXT_PUBLIC_LOGROCKET_APP_ID=your_app_id
   ```

## Usage

See `src/lib/logrocket.ts` for the error tracking client implementation.
